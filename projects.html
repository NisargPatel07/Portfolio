SOCLite
SOC Detection & Response Platform
Problem

Security teams need fast detection from login logs. Manual review is slow and error-prone.

Solution

Ingest auth logs, run detection rules, generate prioritized alerts, and track triage in a dashboard.

Key Detections
  • Brute force (many failures for one user from one IP)
  • Password spray (one IP failing across many users)
  • Impossible travel (country jump too fast)
Stack
Python • Pandas • Flask
Architecture
Auth Logs (CSV)
  -> Rule Engine
  -> Alert Builder (severity + summary)
  -> Alerts JSON
  -> Flask Dashboard (triage + close)
Incident Forecaster
Cloud Incident Prediction Platform
Problem

Outages cost money. Teams want early warnings before failures happen.

Solution

Train a model on metrics patterns and score each host with an incident risk value.

Signals
  • CPU, memory, disk, network
  • Error counts (log-derived)
  • Rolling averages and volatility features
Stack
Python • Pandas • scikit-learn • Flask
Pipeline
Metrics CSV
  -> Feature Engineering
  -> Train Model
  -> Score Latest Host State
  -> Risk Dashboard
System Health Monitor
Monitoring + Ticketing Lab
Problem

IT teams need visibility and a lightweight way to track incidents and recurring failures.

Solution

Monitor system health, auto-create tickets when thresholds are crossed, and keep history for review.

Capabilities
  • CPU + disk monitoring
  • Auto-ticket creation
  • SQLite persistence
Stack
Python • Flask • SQLite
Architecture
Background Monitor
  -> Threshold Check
  -> Ticket DB (SQLite)
  -> Dashboard (Flask)
